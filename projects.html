EcoPickup – Smart Waste Management System

EcoPickup is a smart waste management platform designed to promote cleaner and greener communities. The system allows users to schedule waste pickups, track the status of their requests in real time, and monitor their environmental impact. It also provides an admin panel to manage pickup requests and monitor overall system performance.

Technologies Used: HTML, CSS, JavaScript, Node.js, Express.js

Key Features

  • Waste pickup scheduling
  • Real-time request tracking
  • Environmental impact monitoring
  • User dashboard
  • Admin panel for request management
  • Rewards and achievement system

BookMySlot – Wi-Fi Event Booking System

BookMySlot is a web-based event booking system developed for colleges to simplify event registrations. It enables students to browse events and reserve seats through the campus Wi-Fi network, while administrators can create events, manage registrations, and monitor bookings in real time.

Technologies Used: HTML, CSS, JavaScript, Firebase Authentication, Firebase Firestore

Key Features

  • Secure login using Firebase Authentication
  • Event creation and management
  • Real-time seat availability tracking
  • Student event booking
  • Admin dashboard
  • Centralized booking database
